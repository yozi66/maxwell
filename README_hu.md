Maxwell-egyenletek
==================

A Maxwell-egyenletek differenciális alakja:

```math
\begin{align}
\vec{\nabla} \cdot \vec{D} &= \rho \\
\vec{\nabla} \cdot \vec{B} &= 0 \\
\vec{\nabla} \times \vec{E} &= - \frac{\partial}{\partial t} \vec B \\
\vec{\nabla} \times \vec{H} &= \frac{\partial}{\partial t} \vec D + \vec{j}
\end{align}
```

Nézzük meg egyesével

1. Gauss-törvény
----------------

```math
\vec{\nabla} \cdot \vec{D} = \rho
```

Jelentése: Az elektromos tér forrása az elektromos töltés. A $\rho$ az elektromos töltéssűrűség. Az elektromos erővonalak a pozitív töltésekről
indulnak és a negatív töltéseken végződnek. 


2. Gauss mágneses törvénye
--------------------------

```math
\vec{\nabla} \cdot \vec{B} = 0
```

Jelentése: Nincsenek mágneses monopólusok. A mágneses erővonalak önmagukba záródnak. 


3. Faraday-lenz-törvény
-----------------------

```math
\vec{\nabla} \times \vec{E} = - \frac{\partial \vec{B}}{\partial t}
```

A mágneses indukció változása örvényes elektromos teret indukál, amelynek iránya ellenkező, mint az őt létrehozó változás. 

4. Ampère-törvény
-----------------

```math
\vec{\nabla} \times \vec{H} = \frac{\partial \vec{D}}{\partial t} + \vec{j}
```

Az elektromos áram és az elektromos tér változása mágneses teret hoz létre.
